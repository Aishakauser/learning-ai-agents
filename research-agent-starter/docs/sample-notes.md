## Overview
Project Lighthouse is our internal initiative to build an AI assistant for the
operations team. It is not public and is not mentioned anywhere online — so if the
agent can answer questions about it, that answer came from THIS file via RAG, not
from the model's training data.

## Key facts
- Official launch date: **March 14, 2027**.

## Goals
1. Cut manual report preparation time by 40%.
2. Automate weekly status summaries for the operations leadership.
3. Provide a searchable knowledge base over internal runbooks.

## Constraints
- Must keep all data on internal systems; no customer data leaves the company.
- Phase 1 covers English only; Arabic and Hindi support is planned for Phase 2.
